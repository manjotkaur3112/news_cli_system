# 📰 News CLI System

A simple **Command Line Interface (CLI)** project built with Python to manage news records. Users can add, view, search, and delete news directly from the terminal.

---

## 📌 Features

* ➕ Add new news
* 📄 View all news
* 🔍 Search news
* ❌ Delete news
* 🚪 Exit the application

---

## 📂 Project Structure

```text
news_cli/
│
├── main.py          # Main menu
├── data.py          # Functions for news operations
├── news.csv         # Stores all news data
└── README.md
```

---

## ▶️ How to Run

1. Make sure Python is installed.
2. Open the project folder in the terminal.
3. Run the following command:

```bash
python main.py
```
---

## 💾 Data Storage

All news records are stored in a **CSV file**.

Example:

| ID | Category   | Title            | Description                           |
| -- | ---------- | ---------------- | ------------------------------------- |
| 1  | Sports     | India Wins Match | India defeated Australia by 5 wickets |
| 2  | Technology | AI Update        | New AI model released                 |
